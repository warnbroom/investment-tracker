Công cụ · Debug

Test proxy

Trang này giúp debug proxy Cloudflare Worker. Mỗi bước hiển thị kết quả độc lập nên dễ tìm lỗi.

Bước 1 — Dán URL Worker

URL phải có ?url={URL} ở cuối. Chữ {URL} viết đúng y hệt, cả dấu ngoặc nhọn.

Bước 1b — Kiểm tra Worker có sống không

Gọi URL Worker mà không kèm tham số, kiểm tra Worker có phản hồi. Nếu bước này fail, sẽ không có bước nào khác chạy được. 


    


    
    

      

        Bước 2 — Test raw fetch tới DOJI
      

      

        Gọi thử https://giavang.doji.vn/ qua proxy và hiển thị kết quả thô.
      

      
      

    


    
    

      

        Bước 3 — Parse bảng giá DOJI
      

      

        Dùng kết quả bước 2 để parse thành bảng các loại vàng và giá.
      

      
      

    


    
    

      

        Bước 4 — Test Fmarket API
      

      

        Gọi thử API Fmarket (POST JSON) qua proxy — đây là phần dễ lỗi nhất nếu Worker không hỗ trợ POST.
      

      
      

    


    
    

      

        Bước 5 — Lưu proxy
      

      

        Nếu bước 2-4 đều OK, bấm Lưu để hệ thống nhớ URL này. Lần sau không phải dán lại.
